# Prompt Design

Prompts encode workflows that users intentionally invoke.

Good prompts:

- Ask for a goal.
- Ask for constraints.
- Tell the model what to inspect.
- Tell the model what output to produce.

Prompt examples in this template cover server design, design review, and adding one tool.

Keep prompts focused. If a workflow needs computation or validation, use a tool inside the workflow rather than embedding hidden logic in prompt text.
